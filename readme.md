# DzSteamleet

Простий Streamlit чатбот з Gemini / LangGraph, який управляє задачами.

## Опис

Цей проєкт використовує:
- `Streamlit` для веб-інтерфейсу
- Gemini / `google-genai` для генерації відповідей
- `langgraph` + `langchain-google-genai` для побудови агента з інструментами
- локальний менеджер задач, синхронізований між агентом і UI

## Файли

- `app.py` — основний Streamlit додаток
- `agent.py` — реалізація LangGraph агента та інструментів
- `requirements.txt` — залежності Python
- `.gitignore` — файли, які не варто комітити

## Встановлення

1. Створіть та активуйте віртуальне оточення:
   ```bash
   python -m venv .venv
   .venv\\Scripts\\activate
   ```

2. Встановіть залежності:
   ```bash
   pip install -r requirements.txt
   ```

3. Додайте `GOOGLE_API_KEY` у `secrets.toml` або як змінну середовища.

## Запуск

```bash
streamlit run app.py
```

## Налаштування ключа

У файлі `.streamlit/secrets.toml` додайте:

```toml
GOOGLE_API_KEY = "ваш_ключ"
```

Або встановіть змінну середовища перед запуском:

```bash
set GOOGLE_API_KEY=ваш_ключ
```

## Функціонал

- Чат з Gemini або звичайний режим без інструментів
- Автоматичне створення, відмітка та перелік задач через агента
- UI для швидкого додавання та фільтрації задач
- Канбан-панель для перегляду статусів задач

## Корисні поради

- Не комітіть `.venv/`, `secrets.toml` або інші локальні файли.
- Якщо використовуєте Windows, активуйте віртуальне оточення командою `.venv\\Scripts\\activate`.
- Щоб оновити залежності, повторно запустіть `pip install -r requirements.txt`.

## Ліцензія
